望月ぺろさんの件について、警察の判断は「事故」でした。  
それで手続きが進むのも分かっています。けれど、私にはどうしても飲み込めない点があります。

私は、ぺろさんが毎晩の戸締りの最後に、二階テラスの“決まった手すり”に身を預けて煙草を吸うことを知っています。  
あれは癖というより、あの人なりの区切りでした。  
だからこそ、あの場所で起きたことを「よくある事故」として終わらせるのは無理です。

いま手元にある情報は断片ばかりです。  
それでも、現場にいた私たちにしか拾えない何かがあるはずです。

このサイトは、その断片を整理して保管するために作りました。  
検索ボックスにキーワードを入力すると、こちらでまとめた資料が開きます。  
（必要以上に難しくするつもりはありません。探し方も、追って載せます。）

お願いがあります。  
ぺろさんの最後の夜について、あなたが見たこと・聞いたこと・覚えていることを、ここに集めさせてください。  
感情論で誰かを責めたいわけではありません。  
ただ、事実を揃えたい。揃えないと、次に進めない。

その勢いに、私がついていけなかっただけかもしれません。
